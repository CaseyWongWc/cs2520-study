# Participation Activity 11.7.1 A few standard library modules.
Match the program behavior to a standard library module used to implement the desired program.
1) A trivia game generates a new question at random time intervals.
Answer: random
2) Retrieve the contents of the webpage zybooks.com.
Answer: urllib
3) Get the name of the current operating system.
Answer: os
4) Compute the mathematical cosine function of a user-entered angle in radians.
Answer: math
