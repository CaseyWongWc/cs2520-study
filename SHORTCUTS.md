# 🎮 Jupyter Notebook Keyboard Shortcuts

> Press **`Escape`** first to enter Command Mode before using cell shortcuts!
> Press **`Enter`** to go back to typing inside a cell.

---

## ▶️ Running Cells (works anytime)

| Keys | What happens |
|------|-------------|
| `Shift + Enter` | Run cell → move to next |
| `Ctrl + Enter` | Run cell → stay here |
| `Alt + Enter` | Run cell → add new cell below |

---

## 📦 Add / Delete Cells (press `Esc` first!)

| Keys | What happens |
|------|-------------|
| `A` | Add cell **Above** |
| `B` | Add cell **Below** |
| `D D` (D twice) | **Delete** cell |
| `Z` | **Undo** delete |

---

## 📝 Change Cell Type (press `Esc` first!)

| Keys | What happens |
|------|-------------|
| `M` | 💬 Make it **Markdown** (notes/text) |
| `Y` | 🐍 Make it **Code** (Python) |

---

## 🔍 Navigate

| Keys | What happens |
|------|-------------|
| `↑` / `↓` | Move between cells |
| `Ctrl + Shift + P` | Open command palette |

---

## ⚡ Fastest combo ever
`Esc` → `B` → `Enter` = new code cell below, ready to type!
